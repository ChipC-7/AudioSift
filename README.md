# AudioSift 🎵

简洁优雅的视频音频提取工具，从视频中快速提取高质量音频。

## ✨ 功能特点

- 🎬 **多格式支持**：MP4, AVI, MKV, MOV, WMV, FLV, WebM 等
- 🎵 **丰富输出**：MP3, WAV, OGG, M4A, FLAC, AIFF
- 🌓 **主题切换**：深色/浅色模式，自动检测系统主题
- 📊 **实时进度**：可视化进度条，提取状态一目了然
- 🚀 **后台处理**：不卡顿，提取过程中界面流畅响应
- 🖥️ **跨平台**：Windows, Linux 原生支持

## 📦 安装运行

### Ubuntu / Debian
```bash
sudo apt install python3-pyside6 ffmpeg
python3 main.py

🎯 使用指南
1.选择视频：点击"浏览..."选择要处理的视频文件
2.设置输出：选择格式（MP3/WAV/FLAC等）和质量（128k-320k）
3.开始提取：点击"✨ 开始提取音频"
4.查看结果：输出文件保存在视频同级目录
📋 系统要求
Python 3.8+
PySide6
ffmpeg
